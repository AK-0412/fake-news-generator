# Fake News Generator

This is a simple web project that generates random fake news headlines. I created this project to practice basic web development using Python and Flask.

The program randomly selects words from different lists like subjects, actions, objects, and locations, and combines them to create a fake news headline. Every time the page is refreshed or the button is clicked, a new headline is generated.

This project helped me understand how Python can be connected with HTML to build a simple web application.

## Features

- Generates random fake news headlines
- Simple and easy-to-use web interface
- Built using Python and Flask
- Beginner-friendly project

## Technologies Used

- Python
- Flask
- HTML

## How to Run the Project

1. Clone the repository

git clone <your-repository-link>

2. Go to the project folder

cd fake-news-generator

3. Install Flask

pip install flask

4. Run the application

python app.py

5. Open your browser and go to

http://127.0.0.1:5000

## Purpose of the Project

The goal of this project was to practice basic backend development and understand how Flask connects Python code with HTML templates.
